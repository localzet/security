# Roadmap

## Phase 1

- auth core skeleton
- tenant and application domain model
- password hashing service
- OIDC discovery and JWKS endpoints
- CI and repository policy baseline

## Phase 2

- persistence traits and storage adapters
- authorization code flow domain
- session lifecycle management
- refresh token family rotation
- audit sink implementation

## Phase 3

- admin APIs
- device flow for CLI
- passkeys and MFA enrollment
- policy engine contracts
- self-hosted bootstrap and migrations

## Phase 4

- SCIM and SAML bridge
- federation trust model
- advanced risk engine
- signing key rotation service
- release signing and SBOM pipeline
